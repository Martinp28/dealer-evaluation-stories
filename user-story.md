# User Story - Dealer Evaluation System

## Title
Evaluación de precios de distribuidores

## As a
Cliente del sistema

## I want
Comparar precios entre diferentes distribuidores para un producto seleccionado

## So that
Pueda elegir la mejor opción en precio y disponibilidad

---

## Acceptance Criteria
- Mostrar lista de productos disponibles
- Permitir seleccionar un producto
- Mostrar distribuidores asociados
- Mostrar precios por distribuidor
- Permitir ver todos los distribuidores a la vez

---

High

Done
